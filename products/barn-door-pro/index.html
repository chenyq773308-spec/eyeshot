透光门系列不再只是单一移门产品，而是把热熔天然石作为门片、隔断与光感界面的完整产品族。它适用于豪宅家居、会所、样板间、商业展示与酒店公共空间，在不同开合方式中保持同一套材料语言。

系列构成

产品板块谷仓门Plus / 旋转门 / 平开门 / 平移门 / 折叠门 / 综合应用
材料结构热熔天然石复合门片，依据应用场景匹配玻璃、灯光、五金与边框系统
空间用途客餐厅、书房、衣帽间、玄关、展示间、公共接待区与商业橱窗
设计重点以天然石纹承担主视觉，以透光关系强化空间层次，并根据门型控制开启尺度与边框比例
01 / PLUS

谷仓门Plus

谷仓门Plus，移门轻盈开合，安装便捷，天然石透光画面感强。

02 / PIVOT

旋转门

适合玄关、会所入口与展示空间，以中轴或偏轴开启方式呈现更强的仪式感，让门片成为进入空间前的第一道光感界面。

03 / HINGED

平开门

用于卧室、书房、衣帽间等需要明确边界的空间。石纹、边框与合页系统共同控制厚度感，让门在关闭时也像一块完整的透光立面。

04 / SLIDING

平移门

面向长走廊、套房与开放式厅区，通过轨道系统释放墙面空间。透光天然石在移动过程中形成连续的光影变化。

05 / FOLDING

折叠门

适合需要灵活开合的大面宽空间。多扇门片折叠收纳后保留通透尺度，展开时则形成连续石纹屏风。

06 / INTEGRATED

综合应用

当门、墙、展示柜与灯光系统被统一处理，透光天然石可以从单个门片延展为完整空间背景，服务接待、展示、会客与生活场景。
